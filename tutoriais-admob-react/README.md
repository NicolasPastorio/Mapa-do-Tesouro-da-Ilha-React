# Tutoriais: Google AdMob e Site React.js

Este pacote contém tutoriais completos em português brasileiro para criar uma conta no Google AdMob e desenvolver um site profissional em React.js com todas as páginas necessárias para atender aos requisitos do AdMob.

## Conteúdo do Pacote

### 1. `tutorial_admob.md`
Tutorial detalhado sobre como criar uma conta no Google AdMob, incluindo:
- Processo passo a passo de criação da conta
- Verificação de identidade
- Configuração de pagamento
- Requisitos e políticas importantes

### 2. `tutorial_react_site.md`
Tutorial completo para desenvolvimento de um site React.js profissional, incluindo:
- Configuração do ambiente de desenvolvimento
- Criação de todas as páginas necessárias para AdMob:
  - Página inicial (apresentação profissional)
  - Política de privacidade (essencial para AdMob)
  - Página de contato
  - Lista de aplicativos
- Implementação de roteamento
- Estilização com Tailwind CSS
- Instruções de deploy

### 3. `exemplo-pratico/`
Pasta contendo um exemplo prático funcional do site React.js implementado seguindo o tutorial.

## Como Usar

1. **Para AdMob**: Siga o `tutorial_admob.md` para criar sua conta no Google AdMob
2. **Para o Site**: Siga o `tutorial_react_site.md` para desenvolver seu site profissional
3. **Exemplo Prático**: Consulte a pasta `exemplo-pratico/` para ver a implementação completa

## Requisitos

- Node.js (versão 14 ou superior)
- npm ou yarn
- Editor de código (VS Code recomendado)
- Conta do Google (para AdMob)

## Observações Importantes

- **Política de Privacidade**: É obrigatória para usar o Google AdMob
- **Informações Reais**: Substitua todas as informações de exemplo pelas suas informações reais
- **Personalização**: Adapte o design e conteúdo conforme sua necessidade
- **Testes**: Sempre teste o site em diferentes dispositivos antes de publicar

## Suporte

Estes tutoriais foram criados para serem autoexplicativos e completos. Siga os passos cuidadosamente e você terá um site profissional pronto para uso com o Google AdMob.
